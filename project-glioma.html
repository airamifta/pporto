Multi-Strategy Feature Selection for Glioma Grading

Using Classical and Deep Models

Project Summary

This research proposed a novel multi-strategy feature selection pipeline that combines classical methods (like statistical tests) with deep learning models to enhance the performance of glioma grading from medical imaging data. We evaluated multiple classifiers on both the original imbalanced dataset and a balanced dataset created using SMOTE, providing comprehensive metric comparisons to validate the effectiveness of our approach. This work is currently under review for publication.

Key Information

  • Role: First Author, Researcher
  • Tools: Python, Scikit-learn, Deep Learning, SMOTE
  • Status: Published
  • Authors: A. R. M. Rizqi, et al.
  • Publication: IEEE Xplore
  • Deployment: Streamlit

The Research Process

1. Feature Engineering

Extracted a wide range of features from glioma medical images using both classical and deep learning-based techniques.

2. Multi-Strategy Selection

Designed and implemented a pipeline to select the most informative features by combining the strengths of different selection methods.

3. Data Balancing

Applied the SMOTE (Synthetic Minority Over-sampling Technique) to address class imbalance in the dataset, creating a balanced version for training.

4. Comparative Evaluation

Trained and evaluated multiple classifiers (e.g., SVM, Random Forest) on both imbalanced and balanced data, comparing performance with comprehensive metrics.

Results & Learnings

Key Outcomes

The proposed multi-strategy feature selection pipeline demonstrated a significant improvement in glioma grading accuracy compared to using single strategies. The results confirmed that combining classical and deep features, along with proper data balancing, leads to more robust and reliable classification models in medical imaging.

Lessons Learned

This research provided deep insights into the challenges of working with high-dimensional and imbalanced medical datasets. I learned that an intelligent feature selection process is paramount for building effective diagnostic models. It also reinforced the value of a comprehensive evaluation framework to rigorously validate research findings.
